Este proyecto consiste en un Sistema de Gestión Web para la Venta de Frutas y Verduras, diseñado para optimizar tanto la experiencia de 
compra del cliente como la administración del negocio. La plataforma se construyó utilizando PHP para manejar toda la lógica del lado del servidor
y las interacciones dinámicas. Para la persistencia y gestión de la información, se utiliza MySQL como base de datos, lo que permite almacenar y 
recuperar de forma eficiente el catálogo de productos y los detalles de los pedidos. La funcionalidad central del sistema es su robusto CRUD (Crear, Leer, Actualizar y Eliminar), que 
permite a los administradores una gestión completa del inventario: desde crear nuevos productos, visualizar todo el catálogo, hasta editar 
precios o descripciones, y eliminar artículos agotados. En el frontend, se emplean HTML5 y CSS3 para garantizar una interfaz de usuario clara y funcional.
